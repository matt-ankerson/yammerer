Backend server for Yammerer buit with node.js and mongo db.

## Installation

 - Yammerer use [Koajs][1] so you need to [install node][2] v7.6.0 (at least).
 - [Download][3] Mongo Db

  [2]: https://nodejs.org/en/
  [3]: https://www.mongodb.com/download-center#community
